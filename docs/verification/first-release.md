## 免费数据实测

在本机网络环境中完成了以下真实请求烟雾测试：

- A 股 `600000`：AKShare A 股日线正常返回。
- 港股 `00700`：主接口网络失败后，`stock_hk_daily` 免费备用接口正常返回并按日期截取。
- 沪深 300 ETF `510300`：主接口网络失败后，未复权请求由 `fund_etf_hist_sina` 免费备用接口正常返回。
- 场外开放式基金 `000001`：`fund_open_fund_info_em` 单位净值接口正常返回。

回退规则是显式的：ETF 的备用接口仅用于未复权数据，系统不会把前复权或后复权请求静默降级；接口空数据、字段变化或认证失败都会抛出错误。

## 数据与研究边界

- A 股和港股第一版以日线/周线研究为主，未提供逐笔、Level-2 或券商交易接口。
- 场外基金净值使用独立字段契约，不伪装成 OHLC 行情。
- 同一标的缓存不能混合不同复权模式；同年度增量写入会合并、按日期去重并重建完整清单摘要。
- 回测信号在收盘后生成，最早于下一可交易时点成交，并计入费用和滑点。
- GitHub CodeQL 工作流采用手动触发；私有仓库能否运行取决于账户的 GitHub Code Security 权限。
