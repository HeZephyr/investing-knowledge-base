# 公开投资框架目录

这里只收录可回到原始作品、能拆成可检验方法的公开材料；入选不构成认可、荐股或业绩背书。人物观点属于 C 级解释性来源，估值数据、因子构造和公司法定报告可按其原始机构另行评级。

- [Berkshire 股东信](cards/berkshire-letters.md)：企业所有者视角、资本配置、会计与长期沟通。
- [Howard Marks Memos](cards/howard-marks-memos.md)：周期、风险、价格与投资者心理。
- [Damodaran 估值资料](cards/damodaran-valuation.md)：现金流、风险、增长与叙事—数字一致性。
- [Kenneth French Data Library](cards/kenneth-french-data-library.md)：因子定义、组合构造、历史档案与数据变更。
- [Shiller / Yale《金融市场》](cards/shiller-yale-financial-markets.md)：风险管理、金融制度与行为机制；2011 年静态课程不代表当前规则。
- [AQR Data Library](cards/aqr-data-library.md)：论文关联的假设因子组合、月度/日度序列与修订风险；受现行网站条款约束，只存链接和元数据，不再分发工作簿。
- [GMO Research Library](cards/gmo-research-library.md)：长期估值、均值回归和资产配置观点；机构利益与专有模型必须单独披露。

使用这些材料时同时记录利益关系、适用市场、样本期、后来修订和失败条件。优先提炼问题清单与复现方法，不收集名言。
