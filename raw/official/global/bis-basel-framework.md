# Basel Framework

## 权威性与用途

Basel Framework 汇总巴塞尔银行监管委员会关于资本、风险加权资产、杠杆率、流动性、大额风险和披露的全球标准。它用于理解 CET1、RWA、LCR、NSFR 和 Pillar 3 的概念边界，不直接替代任何司法辖区的落地规则。

## 更新频率与字段

框架按标准和章节组织，并为当前及未来版本标注生效日期。关键模块包括 CAP、RBC、CRE、MAR、OPE、LEV、LCR、NSF、LEX、SRP 与 DIS。研究时记录章节编号、发布日期、生效日和适用银行范围；不能只写“符合巴塞尔 III”而不指明版本。

## 许可与使用边界

BIS 网站提供公开阅读和规范引用入口。本仓库只链接、概括概念并记录版本，不复制完整标准或大段表格。引用、翻译或再利用前应核验 BIS 当期版权及使用条件。

## 利益冲突

BCBS 是审慎监管标准制定者，不销售证券；其目标是银行体系稳健，而不是解释股东回报。监管指标会影响银行行为，使用者必须区分监管安全边界、会计价值与股票估值。

## 历史修订

同一章节可能同时存在当前、未来和已修改版本，成员经济体也在不同日期实施。研究必须以“当时当地已生效”的本地规则为准，并记录过渡安排；用当前合并框架回填历史会产生制度时点偏差。

## 局限与失效条件

Basel 是全球最低框架，国家监管者可作本地定义、比例适用和过渡安排；普通银行、保险公司和券商的资本口径也不相同。找不到本地实施文本、合并范围或资本定义时，不比较跨国 CET1/RWA，也不据此判断估值便宜。
