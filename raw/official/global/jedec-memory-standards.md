# JEDEC 存储标准

DDR、HBM、LPDDR、UFS、e.MMC 与 SSD 等微电子标准入口。用于统一技术术语和代际关系；标准可见性与下载许可不同，不在仓库复制受限标准全文。标准发布不能证明某家公司已量产、通过客户认证或取得经济回报。
