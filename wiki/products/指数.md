# 指数

指数按公开或许可的方法把一组证券映射为点位/收益序列。它不是自然存在的“市场”，编制范围、资格、选样、自由流通、权重上限、调样、公司行动、价格/全收益和币种口径共同决定暴露。

## 方法论拆解

1. 母样本与可投资性：市场、证券类型、流动性、自由流通和外国投资限制。
2. 选择与权重：市值、自由流通市值、等权、价格、基本面或因子；上限规则会产生再分配。
3. 维护：定期评审、快速纳入/剔除、停牌、退市、并购、分拆和股份数变化。
4. 收益口径：价格、总收益、净总收益、超额收益、币种与套保规则。

非市场驱动的成分变更不应凭空改变指数点位。除数连续性练习：

```python
from investkb.assets import index_divisor_after_rebalance

continuity = index_divisor_after_rebalance(1_000, 10, 1_200)
```

方法文件、成分生效日、股份数、自由流通因子、汇率与公司行动都要冻结。指数供应商可能修订历史文件，不能只保存今天的成分表。

指数本身通常不能直接买卖，ETF、期货和互换会引入费用、税、跟踪、融资与交易误差。用指数作为 [[基准]] 前，应确认市场、投资者可达性、币种、分红、税和再平衡口径一致。参见 [[ETF]]、[[衍生品]]。
