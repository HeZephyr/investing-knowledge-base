# 期权与波动率

看涨期权给买方按执行价买入的权利，看跌期权给买方卖出的权利；卖方承担对应义务。到期收益图先于定价模型：`call=max(S-K,0)`，`put=max(K-S,0)`，再把权利金现金流与多空方向分开。

```python
from investkb.assets import black_scholes, implied_volatility, option_payoff

expiry = option_payoff("call", spot=120, strike=100, premium=6)
model = black_scholes("call", 100, 100, 1, 0.05, 0.20)
iv = implied_volatility("call", model.price, 100, 100, 1, 0.05)
```

## Greeks 是局部敏感度

- Delta：标的小变化的一阶近似；Gamma：Delta 的变化。
- Vega：波动率输入变化的局部影响；不同系统可能按 1.0 或 1 个百分点报告。
- Theta：时间流逝的局部影响；Rho：利率输入敏感度。

Black-Scholes 练习假设欧式、连续交易、常数波动率/利率/股息率、对数正态与无摩擦。真实期权存在波动率微笑、跳跃、早行权、离散分红、交易限制、买卖价差和保证金。隐含波动率是给定模型与价格的反解，不是未来实际波动率预言。

## 验证顺序

先检查无套利上下界与 put-call parity，再看数值收敛、单位和 Greeks；最后才做情景。压力至少覆盖标的跳跃、波动率曲面移动、时间流逝、流动性消失和卖方保证金。

## 常见失败

- 用到期收益图解释到期前价格，遗漏时间价值和波动率。
- 把 Delta 当固定胜率、把 Theta 当每天恒定收入。
- 用单一 IV 比较不同期限/执行价，忽略曲面和事件。
- 期权卖方只看收到的权利金，不看尾部和追加保证金。
