# 可转债

可转债是发行人债务与持有人转换选择权的组合。价格受债底、正股、转股价、波动率、利率、信用、流动性、赎回、回售和转股价调整共同影响。

## 条款地图

- 债务端：面值、票息、到期、顺位、担保与违约回收。
- 转换端：转股期、转股价、转换比例、股份来源和稀释。
- 路径条款：强制赎回、持有人回售、转股价下修及各自观察窗、阈值和董事会选择权。
- 事件时点：条款满足、董事会决定、公告、停止交易、最后转股和资金到账不是同一天。

教学量：`转换比例 = 面值 / 转股价`，`转换价值 = 转换比例 × 正股价`。转股溢价和相对债底溢价回答不同问题，债底本身也是依赖信用与利率假设的估计。

```python
from investkb.assets import convertible_metrics

metrics = convertible_metrics(100, 20, 22, 118, 94)
```

## 常见失败

- 把债底当保本价，忽略违约、流动性和估计误差。
- 只看转股溢价，漏掉即将触发的强赎、回售或下修选择。
- 用今天的转股价回算历史，未保存历次除权调整和公告可得日。
- 把可交换债、定向可转债和公开可转债当成同一权利结构。

上交所 2025 年 3 月监管指引要求持续披露转股、赎回和回售；真实研究必须读取募集说明书和最新公告。参见 [[债券]]、[[期权与波动率]]、[[资本配置治理与稀释]]。
