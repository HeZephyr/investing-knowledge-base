# 复利与贴现

## 学习目标

能够把不同日期的现金流移动到同一个时点，区分名义利率、每期利率和有效年利率，并在看到高收益宣传时先检查频率、费用、税和现金流时点。这里的公式是计量尺，不是预测收益的机器。

## 核心概念

本金 (P) 以每年 (m) 次复利、名义年利率 (r) 增长 (t) 年，终值为 (FV=P(1+r/m)^{mt})。贴现是逆运算：(PV=FV/(1+r/m)^{mt})。有效年利率 (EAR=(1+r/m)^m-1)，它把不同复利频率放到相同一年尺度。

贴现率不是凭空给出的常数。对确定现金流，它至少反映期限和无风险机会成本；对风险现金流，还要处理风险、币种、流动性和情景。把高风险公司的预期现金流用存款利率贴现，会系统性高估价值。名义现金流应配名义贴现率，实际现金流应配实际贴现率，不能混用。

## 手算例

把 1,000 元按年利率 10% 年复利两年：第一年末 1,100，第二年末 1,210。反过来，用 10% 把两年后的 1,210 折回今天，得到 1,000。若广告写“名义年利率 12%，按月复利”，每月 1%，有效年利率是 (1.01^{12}-1\approx12.68\%\)，不能直接把 12% 当最终年收益。

再做一个决策例：今天支付 900，一年后确定收到 1,000，忽略税费的回报率为 11.11%。若你的同期限机会成本是 5%，现值约 952.38，净现值为正；但只要“确定收到”不成立，就必须加入违约情景，不能用同一结论。

## Python 实验

```python
from investkb.education import effective_annual_rate, future_value, present_value

fv = future_value(1_000, 0.10, 2)
pv = present_value(fv, 0.10, 2)
ear = effective_annual_rate(0.12, 12)
assert round(fv, 2) == 1210.00
assert round(pv, 2) == 1000.00
assert round(ear, 4) == 0.1268
```

函数会拒绝 NaN、非法频率和使复利底数不为正的利率。主动测试错误输入比只看正确输出更重要。

## 投资场景

估值、债券、定投、基金费用和公司资本开支都依赖现金流时点。比较两个基金时，不仅看年化收益，还要把申赎费、管理费、分红再投资和税后的现金流放在真实日期上。比较跨币种资产时，还要把汇率现金流单列；人民币终值与美元终值不能在没有汇率假设时直接相加。

## 常见误用

- 把简单平均月收益乘 12 当复合年收益。
- 把名义年利率与有效年利率混为一谈。
- 用同一贴现率处理所有期限、币种和风险情景。
- 忽略现金流在期初还是期末，或把尚未收到的分红当作已实现。
- 用过去 CAGR 暗示未来必然复利；复利公式计算的是给定路径，不提供路径概率。

## 检查清单

1. 现金流币种、日期、税前税后是否一致？
2. 利率是名义、实际、单利还是复利？每年复利几次？
3. 贴现率与现金流风险、期限是否匹配？
4. 费用和再投资假设是否写出？
5. 是否做了低/中/高情景，而不是只报一个精确终值？
