# 抽样与估计

## 学习目标

区分总体、抽样框、样本、参数和估计量；理解标准差与标准误不同；能手算简单均值区间，并知道置信区间不能修复选择偏差、时间依赖或错误的数据口径。

## 核心概念

总体是问题真正关心的对象，抽样框是有机会进入数据的对象，样本是实际观察到的部分。若研究“所有上市股票长期回报”，只下载今天仍上市的成分股，抽样框已经删除退市者，后续再精确的标准误也无法消除幸存者偏差。

样本均值是总体均值的估计量。样本标准差 (s) 描述观测之间的离散；均值标准误 (SE=s/\sqrt{n}) 描述在特定抽样假设下估计量的波动。正态近似区间为 \(\bar{x}\pm z\times SE\)。它依赖独立性、代表性和近似分布；小样本通常应考虑 t 分布，时间序列还需处理自相关和异方差。

“95% 置信”是方法长期覆盖率的陈述，不是给定区间有 95% 概率包含固定参数的频率学派解释。更重要的是，区间只量化模型内抽样误差，不包含数据供应商错误、定义漂移、选择偏差或未来修订。

## 手算例

三个教学观测为 1、2、3，样本均值 2，样本标准差 1，标准误 (1/\sqrt3\approx0.577)。使用 1.96 的正态临界值，区间约为 (2\pm1.131)，即 [0.869, 3.131]。样本极小，这个例子只为核对算术；真实推断不应因为区间可算就忽略 t 分布和数据生成过程。

若把同一股票连续三天收益当三个独立样本，波动聚集会让标准误偏小。增加更多高度相关的分钟数据也不会按 (1/\sqrt n) 提供同等的新信息。

## Python 实验

```python
from investkb.education import mean_confidence_interval

estimate, lower, upper = mean_confidence_interval([1.0, 2.0, 3.0])
assert estimate == 2.0
assert lower < estimate < upper
```

函数明确使用 `ddof=1` 的样本标准差并拒绝单个观测。它没有假装自动选择正确分布或修复自相关；这些假设必须由研究者说明。

## 投资场景

估计基金超额收益、公司毛利率正常区间、分析师预测误差或因子分组收益时，先画出抽样单位：证券、日期、公司季度还是基金经理？再写出纳入/排除规则、数据当时可得日和缺失机制。横截面很多股票并不意味着行业冲击下独立，时间很长也不意味着制度稳定。

## 常见误用

- 把标准差当标准误，或把样本量增加当成偏差消失。
- 把成分股当前列表用于历史，制造幸存者偏差。
- 对自相关时间序列使用独立同分布公式。
- 只报告点估计和小数位，不报告区间与经济量级。
- 把 95% 置信区间解释成这一次有 95% 概率包含参数。

## 检查清单

1. 总体、抽样框、抽样单位和时间范围是什么？
2. 退市、停牌、缺失和数据修订如何处理？
3. 观测是否独立；若否，标准误如何调整？
4. 区间使用哪个分布和临界值，为什么？
5. 统计不确定性之外还有哪些口径和模型风险？
