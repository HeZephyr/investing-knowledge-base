# 债券与利率基础

## 学习目标

能把固定息票债券拆成逐期现金流，计算平坦收益率假设下的价格和 Macaulay 久期，理解价格与收益率反向变化，并区分利率风险、信用风险、再投资风险、流动性和可赎回条款。

## 核心概念

债券不是“固定收益一定不亏”，而是发行人对利息和本金的合同承诺。固定息票债券价格等于每期息票与到期本金的现值之和。票面利率决定现金流，市场到期收益率用于折现；两者相等且其他条件简化时，价格接近面值。

到期收益率是把当前价格与合同现金流连接的内部收益率，隐含持有到期、按同一收益率再投资、无违约等强假设。收益率曲线按期限给出不同利率，真实定价应逐现金流使用对应即期利率。信用债还需考虑违约概率、回收率、契约和流动性溢价。

Macaulay 久期是各现金流时点按现值权重的平均时间；修正久期近似价格对收益率小变动的百分比敏感度。凸性解释线性近似在较大利率变化下的误差。久期不是剩余期限，浮息、含权和信用债的有效久期会随状态改变。

## 手算例

面值 100、年息票 5%、两年到期、每年付息，市场收益率 5%。第一年收到 5，第二年收到 105。价格为 (5/1.05+105/1.05^2=100)。Macaulay 久期为 \([1\times5/1.05+2\times105/1.05^2]/100\approx1.9524\) 年。

若收益率上升，两个现金流折现分母变大，价格下降。用久期做一阶近似时必须说明收益率变化单位：100 个基点是 1 个百分点，即 0.01，不是 1%。

## Python 实验

```python
from investkb.education import bond_price, macaulay_duration

price = bond_price(100, coupon_rate=0.05, yield_to_maturity=0.05, years=2)
duration = macaulay_duration(100, coupon_rate=0.05, yield_to_maturity=0.05, years=2)
assert round(price, 2) == 100.00
assert round(duration, 4) == 1.9524
```

教学函数采用一条平坦收益率、固定息票和无违约现金流，只用于核对基础算术。它拒绝不与付息频率对齐的期限和非法收益率。

## 投资场景

比较货币基金、短债基金、长期国债 ETF 和信用债基金时，先看底层久期、信用、杠杆、流动性与费用，而不是只看“债券”标签。债券基金没有固定到期日，净值会随持仓和赎回变化；持有单只债券到期也仍可能面对违约、再投资、通胀和机会成本。

跨市场研究还要核对计息天数、应计利息、净价/全价、结算、税务与币种。A股可转债含股票期权和赎回条款，不能用纯固定债公式解释全部风险。

## 常见误用

- 把票面利率当当前收益率或预期回报。
- 用一个 YTM 折现整条非平坦曲线而不披露简化。
- 认为持有到期就没有经济损失或机会成本。
- 把久期当精确价格变化，忽略凸性和含权行为。
- 比较不同信用、币种、税务和流动性的收益率而不调整。
- 把债券基金等同于一只固定到期债券。

## 检查清单

1. 合同现金流、频率、计息、到期和含权条款是什么？
2. 使用 YTM、即期曲线还是信用利差，日期和币种是否一致？
3. 久期、凸性、信用、流动性和再投资风险分别多大？
4. 报价是净价还是全价，费用和税如何处理？
5. 哪些简化让教学价格不能直接用于真实交易？
