# 金融

## 定义与边界

金融行业至少拆成银行、保险和券商/资本市场业务。三者都管理风险与期限，但资产负债表、收入确认和监管资本完全不同。本页以银行为核心建立方法，再指出保险和券商的分支；不能用银行的净息差或不良率分析保险公司，也不能把券商交易资产当普通工业库存。

银行的存款和借款是生产资料，贷款与证券是收益资产。对银行而言，“高杠杆”是商业模式的一部分，但资本、流动性、资产质量与期限错配决定这种杠杆能否存续。

## 产业链与商业模式

| 类型 | 核心收入 | 核心负债/资金 | 主要风险 |
|---|---|---|---|
| 商业银行 | 净利息收入、手续费 | 存款、批发融资、资本 | 信用、利率、流动性、操作风险 |
| 保险 | 承保利润、投资收益 | 准备金、保费浮存金、资本 | 定价、赔付、久期、市场风险 |
| 券商/投行 | 经纪、承销、资管、做市/交易 | 客户款、回购、债务、资本 | 市场、交易对手、融资、合规 |

银行分析从法律实体和合并范围开始：上市控股公司、银行子公司与受保存款机构可能不是同一边界。FDIC Call Report 适用于美国受保存款机构；上市公司研究还要回到 SEC 或当地法定披露。

## 指标树与时序

银行利润的简化桥：

`税前利润 ≈ 净利息收入 + 非息收入 - 经营费用 - 信用损失准备`

`净息差 NIM = 年化净利息收入 / 平均生息资产`

`信用成本 ≈ 当期信用损失费用 / 平均贷款`

| 观察层 | 指标 | 时序与陷阱 |
|---|---|---|
| 资金 | 存款增速、无息/低息占比、存款成本、批发融资 | 利率变化后存款 beta 常有滞后；期末余额会掩盖季内压力 |
| 资产 | 贷款增速、结构、收益率、证券组合、久期 | 高增长可能降低承保质量；账面分类影响未实现损益呈现 |
| 信用 | 逾期、不良、核销、拨备覆盖、迁徙率 | 不良确认和核销滞后经济恶化；监管宽限会改变可比性 |
| 资本 | CET1、RWA、杠杆率、压力资本要求 | 分子、分母和过渡安排跨地区不同 |
| 流动性 | 高质量流动资产、LCR/NSFR、未保险存款 | 合规比率不能替代挤兑情景与质押可用性 |
| 效率 | 成本收入比、每员工/网点产出 | 重组费用与数字化投资需要一致口径 |

指标顺序通常是市场价格/融资条件先变，存款结构与净息差随后变化，逾期与不良再滞后，核销和资本消耗最后显现。用当季不良率确认周期底部往往太迟。

保险需另看综合成本率、续保/新业务价值、准备金发展、偿付能力和资产负债久期；券商需另看客户资产、交易量、承销管线、净资本、回购融资与 Level 2/3 估值。只有定义相同才可比较。

## 财务映射

- 贷款和证券的利息收益进入收入，但信用损失通过准备计量；现金流量表对金融机构的解释不同于工业企业。
- 存款成本由利率、客户结构、竞争与产品迁移共同决定；“存款增长”若来自高价定期存款，可能压缩而非提升利润。
- 净息差要拆成资产收益率、资金成本、资产负债规模/组合和天数效应。
- 拨备覆盖高不一定安全：要看底层贷款、抵押品、逾期迁徙、核销政策和经济情景。
- 监管资本不是普通股权益的同义词。CET1 需按监管扣减，RWA 取决于风险权重、模型和本地规则。
- 资产质量差会同时影响利润、资本、融资成本和增长能力，四者要在 [[财务三表]] 中联动。

## 估值

银行常用 P/B 或 P/TBV，但必须与可持续 ROE、资本成本、增长和资本充足度联立：

`合理 P/B（稳定增长近似）≈ (ROE - g) / (股权成本 - g)`

这个式子只用于理解敏感性；当 ROE 受周期峰值、一次性拨备回拨或低估信用成本推动时，输出会误导。盈利估值还需使用正常化净息差和穿越周期信用成本。

保险要区分账面价值、内含价值、承保利润与投资收益；券商要区分经常性手续费、周期性投行业务和高波动交易收入。跨类型用一个 P/E 排名没有经济意义。

情景测试至少改变利率曲线、存款 beta、贷款增长、信用成本、证券损失和 RWA；结果同时观察 EPS、CET1 与有形账面价值，而非只看利润。

## 跨市场差异

- Basel Framework 是全球标准入口，不是当地法律。A 股、港股、美股、韩国及其他地区的资本定义、拨备、存款保险、会计准则和实施日期不同。
- 美国 FDIC 数据粒度高，但只直接覆盖美国受保存款机构；不能机械套到银行控股公司或其他市场。
- 国有/政策性目标、利率形成、房地产与地方融资暴露、财富管理和外币资金来源会改变银行商业模式。
- 证券名称相同不代表权利相同；普通股、优先股、永续资本工具和次级债在损失吸收顺序上不同。
- 先统一报告币种、财年、合并范围和监管口径，再处理 [[汇率风险]] 和估值差异。

## 反例与失败模式

1. **利率上升必然利好银行**：资产重定价快于负债时可能成立，但存款迁移、证券久期损失和信用恶化可反转结论。
2. **低 P/B 就便宜**：市场可能在计入低 ROE、资产损失、资本缺口或治理折价。
3. **不良率稳定即风险稳定**：确认滞后、重组、展期、出售和核销会延后损失显现。
4. **资本率跨国直接比较**：本地实施、模型、过渡安排和合并范围不同会制造假可比性。
5. **高拨备覆盖等于高安全边际**：分子很高也可能对应被低估的高风险贷款分母。
6. **把监管数据当上市公司完整财务**：忽略控股公司、非银行子公司和分红上划限制。

## 研究检查清单

- [ ] 法律实体、合并范围、监管主体和上市证券是否一致？
- [ ] 当地资本、流动性、拨备和存款保险规则的生效版本是否固定？
- [ ] 资产收益率、资金成本、净息差和规模变化能否勾稽？
- [ ] 逾期、不良、核销、拨备和重组贷款是否形成迁徙链？
- [ ] CET1 分子、RWA 分母、过渡安排和压力要求是否可比？
- [ ] 是否模拟存款外流、存款 beta、利率曲线和信用成本的联合冲击？
- [ ] 估值是否使用正常化 ROE，而非单季利润或静态低 P/B？
