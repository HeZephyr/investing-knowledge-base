# 材料与矿业

矿业把地质资源经采矿、选矿、冶炼/销售转成可支付金属。资源、储量、矿石吨数、品位、回收率、权益比例和 payable 金属是不同量；材料公司还需接加工能力、能源和产品溢价。

## 单位桥

本页代码只接受无量纲质量分数 `grade_fraction`：1% 必须输入 `0.01`。`矿石吨 × 品位质量分数 = 含量金属吨`；再乘选矿回收率和冶炼可支付比例，才能得到可支付金属。披露若使用 `%`、`g/t`、`oz/t` 或体积浓度，必须先按矿种与密度确定转换规则，禁止把原始数字直接代入。现金成本分母必须明确是产量、销量、权益量还是副产品抵扣后单位。

```python
from investkb.sectors import mining_unit_economics

economics = mining_unit_economics(1_000_000, grade_fraction=0.01, recovery_rate=0.90, payable_rate=0.95, cash_cost=42_750_000)
```

USGS MCS 提供品种级产量与储量背景；SEC S-K 1300 要求重大矿业项目按资源/储量、qualified person 和技术研究披露。两者都不能替代矿山级采剥比、回收、冶炼条款、特许权费、复垦、许可和资本开支。

## 成本曲线与失败模式

成本曲线先统一 co-product/by-product、现金/维持/全成本、币种、能源、运输、权益量和期间。价格上涨会使低品位资源进入储量，从而改变“储量寿命”。

- 把资源量当经济可采储量，或用金属含量当可销售量。
- 用行业平均成本映射单矿，忽略品位、剥采比、能源和地理。
- 把 `1%`、`1 g/t` 或 `1 oz/t` 都输入为 `1`，造成数量级错误。
- 将一次性高品位区外推全矿寿命，或忽略尾矿/复垦负债。
- 用现货价格直接乘储量估值，漏掉数十年时间、capex、税与风险。
