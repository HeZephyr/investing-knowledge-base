# 消费

## 定义与边界

消费研究关注最终消费者如何购买商品和服务，以及品牌、渠道和供应链如何分配价值。先区分必需与可选，再区分品牌商、制造商、经销/零售商、餐饮/门店和平台。宏观“社会零售”或“零售销售”不是上市公司收入的替代变量。

公司收入桥通常是：

`收入变化 ≈ 销量变化 + 价格变化 + 产品/渠道结构变化 + 并购/汇率影响`

实际计算要用乘法或逐项桥接，不能简单把百分比相加；还要扣退货、折扣、积分和渠道库存变化。

## 产业链与商业模式

| 模式 | 核心单位 | 关键经济变量 | 典型风险 |
|---|---|---|---|
| 品牌/快消 | 件、公斤/升、家庭渗透 | 量价结构、渠道铺货、广告促销 | 私牌替代、原料、渠道库存 |
| 耐用品 | 台/套、更新周期 | 订单、客单价、融资与地产相关 | 周期、促销、售后/保修 |
| 连锁零售/餐饮 | 门店、订单、座位/面积 | 同店、客流、客单、成熟曲线 | 选址、租金、人工、闭店 |
| 电商/平台 | 买家、订单、GMV | 转化、复购、获客、佣金/广告 | 补贴、监管、商家质量 |
| 奢侈/高端 | 客户、单品、地区 | 全价销售、稀缺性、旅游流量 | 品牌稀释、灰市、汇率 |

公司映射要按地区、品类、价位、渠道和客户群拆分。行业总量增长可能被另一渠道或竞争者拿走；上市公司的收入确认还可能发生在向经销商出货时，而不是消费者购买时。

## 指标树与时序

`人口/收入/信心/价格 → 客流与转化 → 件数与客单 → 零售终端销售 → 渠道补库/去库 → 公司出货 → 收入 → 毛利 → 费用与现金流`

| 指标 | 定义纪律 | 时序与误区 |
|---|---|---|
| 宏观零售额 | 地区、名义/实际、季调、商品/门店分类 | 名义增长含价格；advance 数据会修订 |
| 同店销售 | 可比门店集合、满店龄、币种、日历 | 新店不在同店但影响总收入；口径可变 |
| 客流/订单与客单 | 线下/线上去重、取消退货 | 客单上涨可能来自涨价或高价渠道偏移 |
| 量价结构 | 销量、标价、净价、mix | 促销和包装缩量会模糊真实价格 |
| 渠道库存 | 公司、经销商、零售商分别看 | 出货领先终端销售会造成虚假增长 |
| 库存天数/折价 | 成本口径、季节性、在途 | 库存增加可能是备货，也可能是滞销 |
| 获客与留存 | cohort、回收期、贡献利润 | 平台补贴会放大 GMV 但不一定创造价值 |

Census 的美国月度零售数据有 advance、后续月度和年度修订；国家统计局社会消费品零售总额也需读取当期附注、调查范围和名义口径。回测必须按首次发布日期冻结，不得用最终修订序列。

## 财务映射

- 收入：拆销量、净价格、产品 mix、渠道 mix、汇率、并购和会计日历。
- 毛利率：拆定价、原料/运费、产品结构、折扣/清仓、产能利用率和存货跌价。
- 销售费用：广告、平台流量、佣金、门店租金和人工要与新增/留存客户或同店增长对应。
- 库存：品牌商、经销商和零售商库存不是一回事；渠道压货会提前收入并推迟问题。
- 现金转换：比较存货、应收、应付和合同负债；利润增长但经营现金流持续落后需回查。
- 门店经济：成熟店贡献利润、开店投入、爬坡期、租约和闭店成本决定扩张是否增值。

一个简化门店模型：

`单店贡献 = 客流 × 转化率 × 客单价 × 毛利率 - 门店人工 - 租金 - 其他可控费用`

总部费用、折旧、税和开店资本仍需另加，不能把“门店 EBITDA”冒充自由现金流。参见 [[财务三表]] 与 [[盈利质量]]。

## 估值

- 品牌商：正常化利润率、品牌再投资、渠道议价和有机增长决定 P/E 或 DCF，而非只按“消费龙头”标签给溢价。
- 零售/餐饮：同店、成熟店回报、净开店、租赁负债和维护资本共同决定价值；门店数增长不等于每股价值增长。
- 平台：GMV 不是收入，收入不是贡献利润；估值要从 take rate、履约/支付成本、获客与留存落到现金流。
- 周期性可选消费：在需求低谷与利润率低谷做多情景，避免用峰值利润的低 P/E 判断便宜。

反向估值先问当前价格隐含的同店、门店数、毛利率和再投资回报，再检查这些假设是否与市场容量和历史 cohort 相容。参见 [[估值]]。

## 跨市场差异

- 中国社会零售与美国 Census 零售在调查对象、分类、季调、价格处理和修订制度上不同，绝对增速不能直接排名。
- A 股、港股和美股公司在收入确认、经销比例、租赁呈现、会员/积分和分部披露上存在差异。
- 全球品牌还受旅游购物、关税、转移定价和 [[汇率风险]] 影响；报告币种增长可能不同于固定汇率增长。
- 线上渗透、即时零售、经销商体系和平台补贴改变渠道库存与价格可见性。
- 公司财年与自然月宏观数据不同，必须按实际公告可得日与地区收入权重对齐。

## 反例与失败模式

1. **宏观零售增长等于公司增长**：公司地区/品类/渠道不同，且份额可能下降。
2. **涨价代表定价权**：销量、促销和包装变化可能抵消涨价；真正定价权要看量与毛利保持。
3. **新店越多越好**：新店分流老店、选址变差或回收期延长会摧毁资本回报。
4. **库存增加是为旺季备货**：若终端销售、折扣和应付不同步，可能是滞销或渠道压货。
5. **GMV 高增长代表盈利增长**：补贴、低 take rate、履约成本和退货会吞噬价值。
6. **最终修订宏观数据回测**：使用当时不存在的最终值会制造历史时点泄漏。

## 研究检查清单

- [ ] 公司收入按地区、品类、渠道和价位的暴露是否与宏观指标一致？
- [ ] 销量、净价、mix、汇率、并购和日历是否独立桥接？
- [ ] 同店集合、店龄、关店、新店分流和成熟曲线口径是否固定？
- [ ] 公司出货、渠道库存和消费者终端销售是否分别观察？
- [ ] 库存、折价、退货、应收应付和经营现金流是否勾稽？
- [ ] 宏观数据的名义/实际、季调、首次发布日期和修订是否记录？
- [ ] 估值是否包含需求、毛利、门店回报和渠道变化的反例情景？
