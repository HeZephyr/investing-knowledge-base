# 地产开发与运营

先分开发、持有运营、物业服务、REIT 和抵押融资。开发商用土地与建设形成项目存货并承担预售、完工、交付和融资错配；运营商依赖租约、入住率、NOI、资本开支和再融资。

## 项目现金与信用

预售签约、合同负债、监管账户收款、会计收入和可自由使用现金不是同一时点。简化项目桥只用于检查现金去向：

```python
from investkb.sectors import property_project_bridge, real_estate_credit_metrics

cash = property_project_bridge(100, 300, 50, 80, 180, 20, 10, closing_cash=160)
credit = real_estate_credit_metrics(debt=500, cash=100, ebitda=120, interest_expense=40)
```

研究保存土地成本/权益、可售面积、售价、去化、收款、开工、完工、交付、建设成本、税费和受限现金。Census SOC 说明许可、开工、在建、完工与销售是不同统计事件，区域宏观不能直接代表公司项目。

## 常见失败

- 把预售额当收入或利润，忽略退款、监管资金和交付义务。
- 用总债务/总现金掩盖项目层级、受限现金、担保和到期墙。
- 用全国房价解释单城单项目，或把住宅逻辑套到办公/物流。
- 把资产评估和 NAV 当可立即成交价值，忽略资本开支和流动性。
