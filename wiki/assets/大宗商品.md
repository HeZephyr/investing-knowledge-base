# 大宗商品

大宗商品研究连接物理供需与金融合约：原油、天然气、金属、农产品的质量、地点、运输、库存和季节性各不相同；“商品指数”通常持有期货而不是仓库现货。现货价格、近月期货、远月期货、生产商股票和商品基金是不同风险暴露。

本页只用于学习，不构成投资建议；第一版不连接期货账户、不自动下单。

## 新手先知道

- 期货是标准化的未来买卖/现金结算合约，保证金是履约保证并逐日盯市，不是只损失首付款的期权费。
- 基差常写为 `现货 − 期货`；contango 表示远月通常高于近月，backwardation 相反。定义必须注明合约、等级、地点和日期。
- 期货总回报包含现货/价格变化、换月收益、现金/保证金收益与费用；只看现货图不能解释基金回报。
- 库存和产量是聚合量，不自动等于某上市公司的销量、实现价或利润。

## 制度或现金流地图

物理层：产地 → 品质/等级 → 加工 → 仓储 → 运输 → 消费地 → 可交割库存。合约层：交易所 → 合约代码/月 → 乘数/报价单位 → 可交割等级/地点 → 最后交易/通知/交割日 → 初始/维持保证金 → 涨跌停/持仓限额 → 结算方式。

投资产品层再区分单合约、连续合约、期货指数、ETF/ETN、生产商股票和资源国货币。每层现金流、信用、税务和跟踪误差不同；ETN 还含发行人信用，生产商股票含成本曲线、套保、资本配置和治理。

商品之间也不能套用同一库存逻辑：原油存在管输与交割地瓶颈，天然气受区域基础设施影响，金属关注交易所/保税/社会库存，农产品受生长季、天气、质量和年度切换影响，黄金的库存—流量关系又与消耗型商品不同。每个品种先写物理单位转换和质量映射。

储存成本、保险、融资、便利收益和季节供需共同影响持有成本关系；偏离教科书公式可能来自不可储存、库存约束或数据口径，而不自动构成套利机会。

## 数据与历史时点

EIA API 能提供能源数量/价格的频率、单位和地区元数据，但库存可能修订，且商业库存不等于交易所可交割库存。期货数据保存交易所、合约月、乘数、币种、时区、结算/成交价、成交量、持仓量和规则版本。

连续合约必须保存换月规则（日期、成交量/持仓量触发、价差调整方式）与原始合约链。回测不得把回溯调整连续价格直接当实际盈亏；最后通知日和流动性必须早于换月决策可得。

## 可执行练习

```python
from investkb.education import annualized_futures_basis, commodity_roll_yield

basis_rate = annualized_futures_basis(spot=100, futures=103, days_to_expiry=90)
one_roll = commodity_roll_yield(near_contract=100, next_contract=104)
```

这里 `basis_rate` 是简单年化的期货减现货比例，不是无风险套利或预期收益；`one_roll` 只给长仓从远月价格向近月价格收敛的价格代理，未计价差、冲击、保证金利息和曲线变化。练习再用 backwardation 情形检查符号。

## 常见失败模式

- 用现货指数回测实际只能持有的期货产品。
- 忽略换月、最后通知/交割日，形成不可执行或意外交割策略。
- 把 contango 等同必亏、backwardation 等同必赚，忽略现货变化和动态曲线。
- 用今天的合约规格、保证金与持仓限额回填历史。
- 把 EIA 总库存、OPEC 产量或黄金基准直接映射到单家公司利润。

## 研究检查清单

1. 物理品种、等级、地点、单位和可交割条件是否一致？
2. 原始合约链、换月规则、通知/交割日和连续调整是否可复现？
3. 保证金、逐日盯市、涨跌停、容量、价差和冲击是否进入现金流？
4. 库存/产量数据的首次发布时间、修订、频率和单位是否保存？
5. 研究的是商品、期货指数、基金、ETN 还是生产商公司，是否偷换暴露？

## 案例骨架

能源示例命题：“库存意外下降支持近月曲线走强”。预注册 EIA 发布时点、市场预期替代、近远月合约、观察窗和失败阈值；只用首次发布值，在下一个可交易时点计算，报告价差、费用和多个库存口径。反例包括库存下降但需求/炼厂检修或交割地约束导致曲线未走强。公司映射必须另接实现价、产量、套保和财报。
