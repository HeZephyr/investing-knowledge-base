# 新手学习路线

1. 风险、骗局、现金管理与“不懂不买”。
2. 从 [[全球市场]] 框架理解 A 股、港股、美股与韩国股市的规则、费用、披露和汇率。
3. 指数、ETF、基金、分散化和长期回报来源。
4. 公司商业模式、财务三表、盈利质量、估值与行业。
5. Python、数据口径、收益率、复权和基础统计。
6. 基准、回测、样本外验证、交易成本与复盘。
7. 用 [[黄金]] 和 [[存储半导体]] 分别练习跨资产与行业周期研究。
