# 新手学习路线

1. 风险、骗局、现金管理与“不懂不买”。
2. 按 [[金融与统计基础]] 完成复利、概率、抽样、检验、回归、预测、债券、公司金融和论文复现的手算与代码练习。
3. 从 [[全球市场]] 框架理解 A 股、港股、美股与韩国股市的规则、费用、披露和汇率。
4. 指数、ETF、基金、分散化和长期回报来源。
5. 公司商业模式、财务三表、盈利质量、估值与行业。
6. Python、数据口径、收益率、复权、基准、回测、样本外验证、交易成本与复盘。
7. 用 [[黄金]] 练习跨资产研究，再用 [[存储半导体]]、[[能源]]、[[金融]] 与 [[消费]] 比较商品周期、资产负债表和单位经济三类行业。
