# 全球免费数据适配

免费行情适合学习、筛选和可复现研究，不等于有 SLA、法定准确性或再分发权。`YFinanceProvider` 把 yfinance 放在可替换 provider 边界后：公开研究保存标准字段和谱系，重要公司事实仍回到交易所与法定披露。

## 从别人的项目复用什么

- **OpenBB**：采用“标准模型—provider 实现—统一结果”的分层思想；不复制 AGPL 源码，也不把“Free”误读为免注册、免限频或免许可。
- **FinHack**：采用权限、空响应、限频和永久失败要分类的经验；不安装当前明示可能不可运行的版本，不接 Token、数据库或交易终端。
- **yfinance**：采用无 Key 的价格与公司行动便利入口；保留其非官方、个人研究用途、字段漂移和无 SLA 边界。

Star 数只用于发现项目。是否进入适配层取决于许可证、固定 commit、字段定义、认证面、失败行为和离线测试。

## 显式证券登记

同一个数字代码在不同市场会冲突，韩国主板与 KOSDAQ 的 provider suffix 也不同。调用者必须登记 canonical market/symbol、provider symbol、币种和交易所时区，适配器不猜：

```python
from investkb.data.global_provider import GlobalInstrument, YFinanceProvider

instruments = [
    GlobalInstrument("US", "AAPL", "AAPL", "USD", "America/New_York"),
    GlobalInstrument("KR", "005930", "005930.KS", "KRW", "Asia/Seoul"),
]
provider = YFinanceProvider(None, instruments)
bars = provider.daily_bars("US", "AAPL", start, end, adjustment="none")
actions = provider.company_actions("US", "AAPL", start, end)
```

`none` 和 `adjusted` 是两个明确口径；A/HK 的 `qfq/hfq` 名称不跨 provider 硬套。上游没有成交额时 `amount` 保持缺失，禁止用收盘价乘成交量伪造。现金分红与拆股进入独立 action 表，拆股没有货币单位。

## 谱系与失败门槛

每个结果保存 provider、endpoint、provider symbol、请求日期、复权口径、UTC 抓取时间、行数和内容 SHA-256。以下情况直接失败：空表、必需字段变化、多层列歧义、重复日期、非正价格、OHLC 关系错误、负成交量和未登记证券。

PR 测试只使用注入的离线 fixture；实时免费源失败不阻断确定性测试。定时 smoke 只负责尽早发现变化，失败后先分类为网络、限频、字段、许可或永久下线，再决定修复、降级或切换来源。

## 检查清单

- [ ] provider symbol、交易所、币种和时区是否显式登记？
- [ ] 价格是否复权，公司行动是否另存，二者有没有重复计入？
- [ ] 数据截止日、抓取时点、字段、行数和哈希是否可追溯？
- [ ] 法定结论是否回到 SEC、KRX/DART 或发行人披露？
- [ ] 免费源失败是否显式呈现，而不是填零或沿用旧价格？
